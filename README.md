# BuffAlert

A Dalamud plugin for FFXIV that displays in-game warning icons when you or your party members are missing role-specific actions.

## Features

- **Tank Stance** - Warns when tanks are missing their stance
- **Healer Buffs** - Sage Kardia, Scholar Fairy
- **DPS Buffs** - Dancer Partner, Summoner Pet, Reaper Soulsow, and more
- **Consumables** - Food, Free Company buffs, Spiritbond potions
- **Gatherer Buffs** - Mining/Botany/Fishing status effects

## How to Use

Use `/buffalert` or `/ba` to open the configuration window.

Warnings display as a simple icon bar that you can position anywhere on screen. Only you can see the warnings - nothing is sent to other players.

## Configuration

- Enable/disable individual modules
- Zone blacklist for areas like Bozja/Eureka
- Test mode to preview and position the warning bar
- Auto-suppress warnings after a set time

## Installation

Available through the Dalamud plugin installer.
